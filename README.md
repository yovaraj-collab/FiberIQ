FiberIQ - OTDR Viewer

A modern web-based Optical Time Domain Reflectometer (OTDR) trace analysis tool designed for engineers, technicians, and fiber optic professionals. OTDR Viewer provides an intuitive interface for loading, visualizing, analyzing, and reporting OTDR traces directly from a web browser.

«Status: 🚧 Beta Version – Under Active Development»

---

Features

📂 File Support

- Import and analyze ".SOR" OTDR trace files
- Drag-and-drop file upload
- Support for multiple trace comparisons

📊 Interactive Trace Visualization

- High-performance trace rendering
- Zoom and pan controls
- Box zoom functionality
- Multi-trace overlay comparison

📏 Measurement Tools

- Distance measurement between markers
- Loss calculation
- Attenuation calculation
- Interactive A/B marker placement

🔍 Event Detection & Analysis

- Automatic event detection
- Reflective event identification
- Non-reflective loss event identification
- Fiber end detection
- Event location and loss reporting
- Potential fault cause suggestions

🎨 Multiple Themes

- Midnight Lab
- Phosphor CRT
- Amber CRT
- Slate Blue
- Daylight
- Paper / Report Mode

📋 Reporting & Export

- Export trace data to CSV
- Export chart images to PNG
- Generate printable reports
- Event summary tables

⚙️ Acquisition Parameters

Displays key OTDR acquisition parameters including:

- Wavelength
- Pulse Width
- Index of Refraction (IOR)
- Averaging Count
- Trace Resolution
- Fiber Type
- Instrument Information

Usage

1. Open OTDR Viewer in your browser.
2. Click Import SOR or drag-and-drop a trace file.
3. Review the trace visualization.
4. Use markers to measure distance and loss.
5. Inspect detected events and fault indicators.
6. Export results or generate reports.

---

Current Limitations

As this application is currently in beta:

- Some vendor-specific SOR formats may not be fully supported.
- Automatic event classification may require verification.
- Additional OTDR vendor compatibility is being added.
- Advanced bidirectional trace analysis is under development.
